# Library Management System

The Library Management System is a Python program that allows users to manage a library's collection of books. It provides functionality to add books, remove books, display books, search for books, and mark books as borrowed or returned.

## Features

- Add a book: Users can add a new book to the library by entering the book's title and author.
- Remove a book: Users can remove a book from the library by specifying the book's title.
- Display books: Users can view all the books available in the library.
- Search for a book: Users can search for a book by entering a keyword, which is matched against the book's title or author.
- Mark a book as borrowed: Users can mark a book as borrowed by specifying the book's title. The program will update the availability status of the book.
- Mark a book as returned: Users can mark a borrowed book as returned by specifying the book's title. The program will update the availability status of the book.

## Requirements

- Python 3.x
